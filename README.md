Overview

This Power BI project provides a comprehensive analysis of supply chain performance across four main stages: manufacturing, storage, shipping, and consumption. The dashboard transforms raw data into clear, interactive insights that support better operational planning and performance optimization.

Objectives

Analyze performance across all supply chain stages: manufacturing, storage, shipping, and consumption.

Identify inefficiencies, gaps, and improvement opportunities.

Provide a visual, interactive tool to support data-driven decision-making.

Data Used

The dashboard uses datasets related to:

Manufacturing: production volume, manufacturing cost

Storage: storage cost, stock levels

Shipping: shipping cost, delivery performance

Consumption: customer demand, consumption volume

Product categories & financial metrics

Dashboard Features

KPIs for cost, revenue, profit, production, storage, and shipping

Category-level comparisons

Cost vs. revenue analysis

Consumption insights and performance indicators

Slicers for filtering by categor or other dimensions

Clear visuals to support operational and strategic decisions

Tools & Technologies

Power BI for data modeling and visualization

Excel / CSV files for data sources

DAX measures for calculations
